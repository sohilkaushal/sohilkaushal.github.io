Projects

MindPad

MindPad is a complete platform for managing brainwave data and a step towards developing a cloud platform for managaing and delivering insights on brainwave data. The project is supposed to be developed as Mind-as-a-Service (MaaS). Currently the platform supports all the devices manufactured by Emotiv.

Architecture

The main component of the project are:

  • Mind Collector:

    This application is supposed to act as an universal interface for collection of EEG data from various devices. After some data has been recorded it may be uploaded to the MindCloud Service for further analysis. It is designed to run as a local service.

  • Mind Cloud:

    This application is intended to be the long term solution for storage of data. It is designed to be run in the cloud.

These applications use Emotiv's Cortex API as a backend for communicating with the headset.
